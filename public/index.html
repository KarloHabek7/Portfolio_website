Featured Projects

Here are a couple of my most significant projects. I've worked on numerous smaller projects as well, constantly honing my skills.

Coil Winder

High school final project (collab. w/ INETEC) for winding coils used in eddy current probes. Features adjustable speed (potentiometer + foot pedal), turn counting (encoder), LCD display (current/total turns, layer info), and user-selectable counting modes. Significantly aids the manual winding process.

View Image

Mars Vision

GUI application (Python/customtkinter) for browsing NASA Mars rover images (Curiosity, Opportunity, Spirit) using NASA's API. Seminar project for OOP, later enhanced. Features include searching by date/sol, 20+ criteria filtering, parallel image downloading, customizable settings (theme, music, image saving options, API key).
